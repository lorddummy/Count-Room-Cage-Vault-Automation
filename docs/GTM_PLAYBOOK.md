# Go-to-Market Playbook

## Why they buy

- **Regulatory:** Count room is #1 audit focus; discrepancies = fines and license risk.
- **Labor:** 4–6 hours/day on count + data entry; variance investigations 5+ hours each.
- **Legacy:** AMS, CDS, IGT, etc. are 30+ years old (AS/400, green screens, Windows XP).
- **Stickiness:** Once managing their money and reconciliation, swap cost is very high.

**Deal sizes:** $200k–$2M per property (one-time) + $50k–$300k/year (SaaS/support).

---

## Trigger events (buying intent)

1. **New casino opening** — Procurement 12–18 months before open. Watch license applications, construction.
2. **Change of ownership** — New owner often standardizes systems.
3. **Commission audit findings** — “Inadequate count room controls” → must upgrade. Reach out right after enforcement.
4. **New CFO/Controller** — First 90 days often have budget for fixes. LinkedIn alerts.
5. **Tech upgrades** — e.g. “Upgrading to Acres” or “mobile app” → modernizing stack.

---

## Discovery (15 questions)

1. What count room system today? (AMS/CDS/Excel = replace candidates.)
2. How long does soft count take? (&gt;6 hr = inefficiency.)
3. How many count room staff per shift?
4. Average daily variance? (&gt;0.5% = serious.)
5. Last commission audit — any count room findings?
6. How do you investigate variances? (Manual video = we win.)
7. Cashier over/short per month?
8. Vault reconciliation time per day? (&gt;3 hr = broken.)
9. Real-time vault visibility? (Usually no.)
10. How do you submit compliance reports? (Manual = we win.)
11. Ever fined for count room issues? Amount?
12. Annual count room labor cost? (ROI calc.)
13. Fills/credits — digital or paper?
14. If count system goes down — what happens?
15. Budget for count room tech this year?

---

## Demo (30 min)

**Attendees:** CFO, Count Room/Cage Manager, Compliance.

- **1–5 min:** “You said 6 hr count with 4 people ($220k/yr) and a $50k fine last year; we address both.”
- **6–12 min:** Count room flow — scan box, expected meter, counter capture, **variance -$350**, alert, “View Video”, close case with note.
- **13–18 min:** Cage — login, TITO scan/validate/void, drawer balance real-time, shift end variance.
- **19–24 min:** Vault — fill request, approve, scan, escort, balance update; CFO dashboard.
- **25–28 min:** Compliance — one-click NV daily drop report, submit to GCB.
- **29–30 min:** Ask — “90-day pilot $30k; if we don’t show 3x ROI, you don’t pay.”

---

## Pilot (90 days)

- **Fee:** $25k–$50k (credit toward Year 1 if they buy).
- **Scope:** Count room full; cage 2–4 cashiers; vault read-only dashboard.
- **Success metrics:** 30%+ count time reduction; 50%+ variance investigation time reduction; 100% report accuracy; 90%+ user satisfaction.
- **Week 1–2:** Integrations (counter, slot system), deploy devices, train.
- **Week 3–12:** Run parallel with legacy; compare daily.
- **Week 13:** Results deck; ask for full deployment. Target conversion 85–90% if metrics hit.

## Pricing (mid-sized property, ~1,500 slots)

**SaaS (recommended):**

- **Year 1:** $120k (implementation + license + support).
- **Year 2+:** $80k/year.
- Payment: quarterly.

**Perpetual (if required):**

- Upfront $250k; annual support $50k; implementation $60k.

**ROI justification (annual):** Count labor savings ~$55k, variance investigation ~$7.5k, over/short reduction ~$6k, compliance prep ~$9k, avoided fines (amortized) ~$25k → **~$102k benefit**; payback ~14 months, 3-year ROI ~156%.

## Contract highlights

- 3-year minimum for SaaS.
- SLA 99.9% uptime.
- 24/7 support (count runs at 2 AM).
- Data ownership and export to customer.
- “Subject to [jurisdiction] gaming commission approval of vendor.”
