# Legacy Systems (Competition)

## 1. AMS Financial (Everi)

- **Share:** ~25% US casinos. 30+ years; AS/400.
- **Pros:** Integrated with Everi slot; cage, count, vault, accounting; commission-approved.
- **Cons:** AS/400 (dying skills); on-prem only; no cloud/mobile; batch, no real-time; green-screen UI.
- **Pricing:** $150k–$500k + $30k–$80k/yr. Casinos want to replace but fear risk.

## 2. CDS (Casino Data Systems)

- **Share:** ~15%, many tribal. 25+ years.
- **Pros:** Strong in tribal; count, cage, accounting.
- **Cons:** Windows XP; no APIs; no surveillance integration; clunky UI.
- **Pricing:** $100k–$300k + $20k–$50k/yr. Losing share.

## 3. IGT Unified Wallet (ex–GCA)

- **Share:** ~20%, mainly cage. 15–20 years.
- **Pros:** IGT slot integration; cage, markers, check cashing.
- **Cons:** Weak count room; expensive; IGT lock-in.
- **Pricing:** $200k–$600k + $40k–$100k/yr.

## 4. Konami (Synkros)

- **Share:** ~10%; stronger in Asia-Pacific.
- **Pros:** Konami slot integration.
- **Cons:** Limited US; fewer features than AMS; expensive.
- **Pricing:** $150k–$400k.

## 5. Homegrown / Excel

- **Share:** ~20% (small/mid).
- **Pros:** “Free,” customized.
- **Cons:** No docs; macros break; no audit trail; no dual control; fraud risk. Commissions pushing modernization.

## Our gap (opportunity)

1. **Cloud-native, mobile-first** — Legacy = on-prem Windows. We = iPad/iPhone + web.
2. **Real-time variance alerts** — Legacy = next-day. We = within 60 seconds.
3. **AI/pattern detection** — e.g. “Cashier over/short 8 of 10 shifts” — legacy has none.
4. **Surveillance integration** — Legacy = manual. We = one-click “View Video.”
5. **Compliance-as-a-service** — Legacy = export, manual format, email. We = one-click submit to commission.
6. **SaaS pricing** — Legacy = big upfront. We = $80k–$150k/yr, no huge capex.
