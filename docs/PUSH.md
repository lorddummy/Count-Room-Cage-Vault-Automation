# Pushing updates to GitHub

Repo: **https://github.com/lorddummy/Count-Room-Cage-Vault-Automation**

From the project root:

```bash
git add .
git commit -m "Your message"
git push origin master
```

SSH remote:

```bash
git remote set-url origin git@github.com:lorddummy/Count-Room-Cage-Vault-Automation.git
```
