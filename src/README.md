# Source

Implementation of Count Room, Cage, and Vault automation and reconciliation will live here.

**Planned:**

- Soft count service (drop box receipt, currency counter integration, variance calc, bag/tag, reports)
- Cage service (drawer lifecycle, transactions, TITO/validator integration, shift reconciliation)
- Vault service (balance, fills/credits, armored car, cage replenishment, reconciliation)
- Table games inventory (float, fills, credits, drop, win/loss)
- Compliance report generation and submission adapters
- Surveillance integration (Genetec, Milestone) for “View Video” and bookmarks
- Analytics/BI and variance alerting
- Integrations: slot systems (IGT, Aristocrat, Everi, Konami, Acres), table systems, accounting, armored car, TITO and account systems

See `docs/` for architecture, modules, integrations, and tech stack.
